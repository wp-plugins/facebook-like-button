Add the new Facebook Like button to your wordpress blog.

== Description ==

Add the facebook like button for your blog. Change the button layout, use XFBML or iFrame and much more just in one plugin.


== Installation ==

1. Unzip the file
2. Upload it to wp-content/plugins
3. Activate it from the plugins section
