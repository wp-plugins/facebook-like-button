== Description ==

Add the new Facebook Like button to your wordpress blog.


== Installation ==

1. Unzip the file
2. Upload it to wp-content/plugins
3. Activate it from the plugins section

== Frequently Asked Questions == 

= Why when I like 1 post it likes all posts? =

This is because you're at the home page and this's a current URL related plugin. So you're not liking all post, you're liking the home page.
